# 🚀 Primeiro Projeto de QA: Testes Funcionais na Plataforma Amazon

Olá! Este é o meu primeiro projeto prático focado na área de Garantia de Qualidade (QA). O objetivo deste trabalho foi aplicar técnicas de **Teste de Caixa Preta** utilizando cenários reais de um dos maiores e-commerces do mundo.

## 🎯 Escopo do Projeto
Focado nas funcionalidades de:
* Mecanismo de busca (Sucesso e Exceção)
* Fluxo de carrinho de compras (Usuário deslogado)
* Manipulação de quantidade e cálculo de valores

## 📁 Artefatos Inclusos
* **Casos de Teste:** Mapeamento completo com passos, resultados esperados e status de execução (disponível no arquivo `Casos_de_Teste.md`).
* **Bug Report:** Demonstração de reporte técnico de falha visual simulada em ambiente mobile (disponível no arquivo `Bug_Report.md`).

## 🛠️ Tecnologias e Metodologias
* Teste Manual Funcional
* Escrita de Casos de Teste (Cenários Positivos e Negativos)
* Ferramentas: GitHub
